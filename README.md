# Vue Auth Dashboard

Projeto base com Vue 3 + Vite + TypeScript, ESLint e Prettier.

## Pré-requisitos

- Node.js 20+
- pnpm (recomendado) ou npm

## Como rodar

1. Instale as dependências:

```bash
pnpm install
```

2. Rode em desenvolvimento:

```bash
pnpm dev
```

3. Build de produção:

```bash
pnpm build
```

4. Preview da build:

```bash
pnpm preview
```

## Scripts úteis

- `pnpm lint` valida com ESLint
- `pnpm format` formata com Prettier

## Estrutura de pastas

```text
src/
  components/
  pages/
  router/
  services/
  stores/
```
